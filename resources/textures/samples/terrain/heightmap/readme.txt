Two example terrains made for demonstration and testing purposes, contains a FPS-map-terrain and a Racing-map-terrain with a racetrack carved in.

Contains 5 terrain textures, available in high resolution source files and gameready exported ones for use with Torque3D, with base, macro, detail and normal maps.

Terrains are also exported to heightmaps with layer masks to quickly load them into other engines.

----------------------

"Torque3D" folder goes to your "game" directory and example terrains will be available in game.

"source" folder contains high resolution source images for the textures and heightmaps with layermaps for import into other engines.
